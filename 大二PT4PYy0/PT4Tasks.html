ZArray113. An array A of N (≤ 6) real numbers is given. Sort the array in ascending order by using the selection sort method: find the greatest element in the array and exchange the values of the greatest element and the last element (with the index N − 1); repeat N − 1 times this process, reducing the amount of analyzed elements by 1 after each array pass. Output all array elements after each pass for checking results.
